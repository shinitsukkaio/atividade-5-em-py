# Atividade 5 - Média de Estudantes

Calcula a média de estudantes por turma (máximo 40 por turma) usando NumPy.

## Como funciona:
- Pede quantidade de turmas
- Coleta número de estudantes (máximo 40)
- Se passar de 40, pede novamente
- Calcula a média com NumPy

## Como usar:
```bash
pip install numpy
python atividade5.py
```

## Exemplo:
```
Quantidade de turmas: 2
Estudantes da (i+1) turma: 35
Estudantes da (i+1) turma: 38
Media de estudantes: 36
```
